# Система управления банковскими картами

Spring Boot приложение с JWT-аутентификацией, базой данных PostgreSQL и Docker

## Возможности (Функциональность)

- Spring Boot 3.x
- Spring Security c JWT
- БД PostgreSQL
- Docker и Docker Compose
- Документация Swagger/OpenAPI

## Быстрый старт

1. Клонировать репозиторий:   
   git clone https://github.com/async0044/bank-rest-app.git
2. Перейти в папку проекта 
   cd bank_rest-main
2. Запустить (docker-compose up --build)
3. Доступ к приложению:
   API: http://localhost:8080
   Swagger UI: http://localhost:8080/swagger-ui.html
   PostgreSQL: localhost:5432

4. Предустановленная учетная запись:
   Username: admin606050
   Password: password606050

5. JWT-токен будет предоставлен в Response после авторизации
